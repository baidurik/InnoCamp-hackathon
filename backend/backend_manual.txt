В папке backend размещены исходники сервера. Чтобы включить сервер, нужно запустить файл application.py
В файле some_useful_functions.py определено несколько полезных функций, необходимых для работы
В файле db_config.py определены средства для работы с базой данных

ВАЖНО: В приложении используется база данных PostgreSQL, которая расположена на сервере, поднятом на личном компьютере бэкендера команды.
       При попытке запустить скрипт работы с БД на другом компьютере, скорее всего выйдет ошибка
